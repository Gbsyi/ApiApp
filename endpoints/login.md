# Авторизация
**URL** : `/api/auth/login`  
  
**Метод** : `POST`  
  
**Необходима авторизация** : Нет  

**Query параметры** : 
| Параметр | Описание            |
|----------|---------------------|
| username | Имя пользователя    |
| password | Пароль пользователя |

## Успешный ответ
**Статусный код** : `200 OK`
**Пример ответа** :  
````
{
    "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVC...",
    "username": "user"
}
````

## Неуспешный ответ
**Статусный код** : `400 Bad Request`
````
{
  message: "Неверные данные"
}
````
