# Регистрация
**URL** : `/api/auth/register`  
  
**Метод** : `POST`  
  
**Необходима авторизация** : Нет  

**Query параметры** : 
| Параметр | Описание            |
|----------|---------------------|
| name     | Имя пользователя    |
| password | Пароль пользователя |

## Успешный ответ
**Статусный код** : `200 OK`

## Неуспешный ответ
**Статусный код** : `401 Bad Request`
````
{
  message: "Неверные данные"
}
````
